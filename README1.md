## ✨ المميزات

- **تصميم متجاوب (Responsive)** يعمل على الجوال والحاسب بنفس الجودة.
- **دعم كامل للغة العربية** واتجاه RTL.
- **زخرفة هندسية** بنجمة ثمانية مرسومة بـ SVG كتوقيع بصري للصفحة.
- **حركات خفيفة** عند التمرير، مع احترام إعداد `prefers-reduced-motion`.
- **إمكانية وصول (Accessibility):** تباين لوني واضح، وتنقّل كامل بلوحة المفاتيح.
- **بدون تبعيات:** لا jQuery ولا Bootstrap — HTML و CSS خالصان.

## 🚀 التشغيل محلياً

```bash
git clone https://github.com/fhggnk163-ops/personal-website.git
cd personal-website
```

ثم افتح ملف `index.html` بأي متصفح — لا حاجة لأي أدوات بناء.

---

## 🌍 النشر على GitHub Pages

1. ادخل إلى المستودع → **Settings** → **Pages**.
2. من **Source** اختر **Deploy from a branch**، ثم الفرع `main` والمجلد `/ (root)`.
3. اضغط **Save**، وبعد دقيقة سيصبح الموقع متاحاً على:
   `https://fhggnk163-ops.github.io/personal-website/`

> ⚠️ يجب أن يكون المستودع **Public** وأن تكون الملفات في **جذر** المستودع.
